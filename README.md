# Best Website Support

Phase A + B scaffold for the Best Website Support plugin.

## Includes in this scaffold
- Hidden settings page (`/wp-admin/admin.php?page=bw-settings`)
- Dashboard cleanup toggles
- Update UI cleanup toggles
- Admin footer branding
- Website Support dashboard widget
- Website Support sidebar page
- Email-based support form with diagnostics metadata

## Next planned features
- Admin restrictions (plugin/theme editor, installs, updates page, etc.)
- Menu cleanup by slug
- Label renaming (Posts/Pages/CPTs)
- Login branding
- White-label plugin visibility
- GitHub updater integration


## Releases & Updates

This plugin is updated via GitHub Releases. Tag a version like `v1.0.1` to trigger the release workflow, which builds `website-support.zip` and publishes a release. Client sites will see updates automatically via the built-in GitHub updater.
